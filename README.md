# SmartAssess

SmartAssess is a web-based online assessment platform designed for academic and training purposes.

## Features
- Student Login & Registration
- Randomized Quiz Questions
- Back, Next & Submit Navigation
- Timer-Based Auto Submission
- Result Analysis with Solutions
- Certificate Generation
- Admin Panel for Question Management

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## Project Type
B.Tech 3rd Year Academic Project
